## CircleLive

**分享，遇见，Live**

分享自己絮语，心情，碎碎念，在Live中遇见共鸣。


## 技术支持
1. Agora互动直播SDK
   > 多人音视频互动
2. Agora云信令
   > 实时消息通信，Live过程的消息分发
3. 环信
   > IM 支持，建立好有关系，会话


## 主要功能
1. 发布和浏览心情动态
2. Live
   * 发布Live
   * 订阅共鸣Live
   * 多人音视频Live
   * Live心情滤镜
3. IM
   * 加好友（自动创建会话）
   * 基本IM（文字，语言，图片，视频，表情）
