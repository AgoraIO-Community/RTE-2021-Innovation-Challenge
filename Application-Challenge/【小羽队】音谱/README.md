**给儿子的礼物中最喜欢的**

## SDK使用

## 主要功能
1. 发布和浏览心情动态
2. Live
   * 发布Live
   * 订阅共鸣Live
   * 多人音视频Live
   * Live心情滤镜
3. IM
   * 加好友（自动创建会话）
   * 基本IM（文字，语言，图片，视频，表情）
