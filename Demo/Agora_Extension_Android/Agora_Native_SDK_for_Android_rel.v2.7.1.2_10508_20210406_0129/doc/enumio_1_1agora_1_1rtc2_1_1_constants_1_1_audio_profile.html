Detailed Description

Sets the audio profile sampling rate, bitrate, encode mode, and the number of channels.

Member Data Documentation

◆ DEFAULT

io.agora.rtc2.Constants.AudioProfile.DEFAULT =(Constants.AUDIO_PROFILE_DEFAULT)

0: The default audio profile.

  • In the Communication profile, the default value is the same with AUDIO_PROFILE_SPEECH_STANDARD(1).
  • In the Live-broadcast profile, it represents a sample rate of 48 kHz, music encoding, mono, and a bitrate of up to 52 Kbps.
